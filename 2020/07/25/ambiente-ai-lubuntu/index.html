Ambiente de Desarrollo para AI

Nuestra Recomendación Lubuntu 18.04

Para trabajar en el programa profesional de inteligencia artificial de UCenfotec, recomendamos instalar una máquina virtual de VirtualBox con Lubuntu 18.04 Bionic Beaver.

Puede descargar los recursos de:

  • Lubuntu 18.04: https://lubuntu.net/lubuntu-18-04-bionic-beaver-released/
  • Virtual Box: https://www.virtualbox.org/wiki/Downloads

Si gusta puede utilizar otros sistemas operativos como Windows o Mac. Sin embargo nuestros instructores estarán utilizando Lubuntu.

Para que pueda utilizar Lubuntu con la resolucion completa de su monitor, recomendamos instalar VirtualBox Guest Additions. El siguiente tutorial le permite instalar los paquetes necesarios.

  • Virtual Box Guest Additions: https://linuxize.com/post/how-to-install-virtualbox-guest-additions-in-ubuntu/

Ambiente de Desarrollo

Durante todo el programa trabajaremos con Python 3.6, este ya viene instalado por defecto en Lubuntu 18.04. Sin embargo no utilizaremos la consola, sino Jupyter Notebooks. Use los siguientes pasos para instalar el gestos de paquetes pip3 y Jupyter.

Gestor de paquetes pip3

  • Abra una terminal
  • sudo apt install python3-pip

Instalar Virtualenv

Virtualenv, es un ambiente virtual para almacenar paquetes de Python. Aqui instalaremos luego keras, tensorflow, y jupyter.

  • Abra una terminal
  • sudo -H pip3 install virtualenv

crear un ambiente virtual para cada curso:

Como el primer curso es de aprendizaje automático, vamos a llamarle ml-env

Abra una terminal y digite el siguiente comando para crearlo

  • virtualenv ml-env

Ahora digite el siguiente comando para activar el ambiente:

  • source ml-env/bin/activate

instalar Jupyter Lab (active ml-env)

Jupyter Lab es una version mejorada de jupyter notebooks. Una vez que instalo Pip3, proceda a instalar el IDE en el ambiente ml-env:

  • pip3 install jupyterlab

para ejecutar jupyter lab, digite en la terminal:

  • jupyter lab

Esto abrirá el navegador en la dirección localhost:8888/lab en donde podrá crear notebooks de Python 3.

Instalar otros paquetes básicos

Abra una nueva terminal:

  • source ml-env/bin/activate
  • pip3 install numpy
  • pip3 install pandas
  • pip3 install matplotlib
  • pip3 install keras
  • pip3 install tensorflow
  • pip3 install scipy

Otros programas

Puede que necesite un editor de texto, puede usar cualquiera de estos:

  • Atom: https://atom.io/
  • Visual Studio Code: https://code.visualstudio.com/
